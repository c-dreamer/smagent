# smagent Control Centre

Run `python3 dashboard/server.py`, then open <http://127.0.0.1:8766>.

The dashboard lists review videos from `~/Downloads/smagent_output`, keeps
publishing disabled, and stores only the Postiz base URL plus the *name* of the
environment variable holding its API key. Set that variable in the shell or
your private environment file before the Postiz publish integration is added.
